# 🏥 Vital Bridge

A comprehensive Healthcare Management System developed using **ASP.NET Core MVC** that helps users access emergency healthcare services quickly and efficiently.

---

## 📌 Project Overview

Vital Bridge is designed to improve healthcare accessibility by providing essential healthcare services through a modern web application. The system enables users to locate nearby hospitals based on their current location, request ambulance services, explore healthcare information, and contact emergency services. It also includes a secure authentication system and a fully functional administrative dashboard for managing the application.

---

## ✨ Features

- 🔐 User Authentication & Authorization
- 🏥 Nearby Hospitals Detection
- 📍 Current Location Support
- 🗺️ Google Maps Places API Integration
- 🚑 Ambulance Services
- 🚨 Emergency Assistance
- 👤 User Management
- 📊 Admin Dashboard
- 📩 Contact & Feedback
- 📱 Responsive Design
